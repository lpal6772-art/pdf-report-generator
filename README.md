# PDF Report Generator using Python

This project reads data from a CSV file, performs analysis, and generates a formatted PDF report.

## Features
- Reads employee data from CSV
- Calculates statistics (average, highest, lowest)
- Department-wise performance analysis
- Generates professional PDF report using FPDF

## Technologies Used
- Python
- FPDF
- CSV module
- Statistics module

## Project Structure
- report_generator.py -> Main Python script
- data.csv -> Input data file
- employee_performance_report.pdf -> Output report (generated)

## How to Run
1. Install Python
2. Install FPDF:
   pip install fpdf
3. Run the script:
   python report_generator.py

## Output
The script generates a PDF report named:
employee_performance_report.pdf
